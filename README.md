👨‍🏫 **프로젝트 개요**
---
**Spring boot, React, Kubernates** 를 활용하여 REST API를 설계 및 'AI를 활용한 북커버 만들기' 웹사이트를 구현했습니다.
또한 OpenAI 통합을 통해 **도서 표지 자동 생성, 요약 생성, 가격 및 카테고리 선정, 오디오북 생성**하는 기능을 포함한
풀스택 웹 애플리케이션을 성공적으로 개발했습니다.

- **목표**:  
  1. 작가 신청 → 심사 → AI 지원 출간 → 구독/결제 → 알림까지  
  2. 전체 워크플로우 자동화 & 이벤트 기반 비동기 연동  
  3. 확장 가능한 마이크로서비스 아키텍처
